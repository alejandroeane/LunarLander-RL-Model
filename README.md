### Objetivos:
Modificar la implementación de la plantilla dada para Cart Pole con el objetivo de generar un modelo mediante DQN que permita operar eficientemente en el entorno Lunar Lander de Gymnasium. El modelo ha de tener una puntuación media superior a 200 y el menor número de pasos posibles durante el entrenamiento.

### Qué hay que hacer:
- Entregar un Notebook Jupyter con el código Python completo correspondiente a la mejor implementación lograda por los estudiantes y que muestre el resultado de la mejor ejecución con la siguiente información:
  
  - Sección de texto que identifique a los autores.
  - Descripciones que expliquen y justifiquen claramente los cambios y extensiones realizados a la plantilla.
  - Número total de pasos para el entrenamiento (es decir, el número total de interacciones con el entorno durante el entrenamiento).
  - La puntuación media (average score) obtenida por el programa de prueba para 30 episodios.
  - La gráfica de la evolución de la puntuación durante el proceso de entrenamiento.
  - Sección de texto que resuma la lista de cambios probados por los estudiantes que produjeron resultados intermedios insatisfactorios.
    
- Entregar el modelo de red neuronal (formato de fichero .keras) final entrenado.
